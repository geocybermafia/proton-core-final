# Hyper-Local Tbilisi Knowledge Graph (Knowledge Base)

This directory contains the knowledge base for Proton-Core's hyper-local AI capabilities.
It uses RAG (Retrieval-Augmented Generation) to ground AI responses in local Tbilisi/Georgian data.

## Structure
- `/knowledge/legal/` - Georgian business law, tax regulations
- `/knowledge/business/` - Local business registration, commercial property, industry reports
- `/knowledge/geo/` - Neighborhood data, infrastructure, transit routes
